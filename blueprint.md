# Project Overview

This project is a simple web application that allows users to generate lottery numbers by clicking a button. The application is built using HTML, CSS, and JavaScript.

# Style, Design, and Features

*   **Design:** It has a clean and modern design.
*   **Color Palette:** It uses a blue and white color scheme, giving it a cool and appealing look.
*   **Font:** It uses system fonts for fast and readable rendering.
*   **Functionality:**
    *   Users can click the "Generate Numbers" button to generate 6 unique lottery numbers.
    *   The generated numbers are displayed in a visually appealing way.
    *   The application is responsive, so it looks good on all devices.
